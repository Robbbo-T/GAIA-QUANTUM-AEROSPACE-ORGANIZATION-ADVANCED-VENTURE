## üéØ Program Objectives

The AMPEL360e represents a revolutionary step toward sustainable aviation, designed as a **zero-emission commercial aircraft** incorporating mature, certifiable hybrid-electric propulsion technologies. This concept phase establishes the foundational framework for developing a next-generation aircraft that meets stringent aerospace certification requirements while delivering superior operational performance.

### Primary Objectives
- **Zero-emission operations** for short to medium-range flights (up to 3,500 km)
- **Hybrid-electric propulsion** integration with mature, certifiable technologies
- **CS-25 compliance** from initial design through certification
- **Q-Division integration** across all technical domains
- **Rapid certification pathway** leveraging existing regulatory frameworks
